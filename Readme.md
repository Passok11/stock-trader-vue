# Stock Trader with Vue

Buy and sell fictitious stocks. be rich or  get bankrupt!

## Getting Started



### Prerequisites

you need node.js and yarn installed

### Installing

for install in your local machine just run:

```
yarn install
```
and for run and test:

```
yarn serve
```
